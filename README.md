# 📋 Sistema de Login e Cadastro de Usuários

## 📌 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar conceitos básicos de desenvolvimento web utilizando **HTML**, **CSS**, **JavaScript** e **Bootstrap**.

A aplicação simula um pequeno sistema com:

- **Tela de login**
- **Tela de cadastro**
- **Validação de campos**
- **Cadastro de usuários em lista**
- **Edição de nomes**
- **Exclusão de usuários**

O foco principal do projeto foi exercitar a manipulação do **DOM**, uso de **funções em JavaScript**, **arrays** e interações com elementos da interface.

---

## 🚀 Funcionalidades

### 🔐 Tela de Login
Na tela inicial, o usuário deve informar:

- E-mail
- Senha

O sistema verifica se os campos foram preenchidos.  
Caso estejam vazios, é exibida uma mensagem de alerta.

Se os campos forem preenchidos corretamente, o usuário é redirecionado para a tela de cadastro.

---

### 👤 Tela de Cadastro
Na tela de cadastro, o usuário pode:

- Informar o nome
- Salvar o nome em uma lista
- Visualizar os usuários cadastrados em tabela
- Editar um nome já salvo
- Excluir um nome da lista
